Bare Bones React provides a modern UI using SASS, Babel, Webpack and more.  

## Documentation

To get started simply download or clone the repository into a folder and open that folder in your terminal. 

> `npm install`

### Development Build

Run the dev server

> `npm run dev`

### Production Build

Build production files

> `npm run prod`
